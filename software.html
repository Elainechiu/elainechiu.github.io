sensitivityIxJ

CRAN: https://CRAN.R-project.org/package=sensitivityIxJ

The sensitivityIxJ R package provides exact, kernel-based, and normal-approximation sensitivity analysis methods for multi-level ($I\times J$) contingency tables under unmeasured confounding. It implements efficient algorithms using Rcpp and supports large-sample approximations, sequential importance sampling, and design sensitivity computation.
