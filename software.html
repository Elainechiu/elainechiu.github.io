sensitivityIxJ: Exact, Nonparametric Sensitivity Analysis for Observational Studies of Contingency Tables

The R package sensitivityIxJ provides exact and nonparametric sensitivity analysis methods for multi-category contingency tables under unmeasured confounding. It generalizes classical binary-treatment sensitivity analysis frameworks to I×J categorical settings.

The package accompanies the paper “Exact, Nonparametric Sensitivity Analysis for Observational Studies of Contingency Tables” by Elaine K. Chiu and Hyunseung Kang.

Source code and documentation are available on GitHub .
